# খেলাঘর লাইভ স্পোর্টস - সার্ভার / ডোমেন হোস্টিং সেটআপ নির্দেশিকা

আপনার নিজের ডোমেন এবং হোস্টিং কিনে অ্যাপের এডমিন প্যানেলটি সক্রিয় করার জন্য নিচের সহজ ধাপগুলো অনুসরণ করুন।

---

## 🛠️ হোস্ট করার সহজ ধাপসমূহ (Step-by-Step Hosting Setup)

### ধাপ ১: ফাইল আপলোড করা
১. আপনার হোস্টিং কন্ট্রোল প্যানেলে (cPanel, DirectAdmin ইত্যাদি) লগইন করুন।
২. **File Manager**-এ গিয়ে `public_html` ফোল্ডারে প্রবেশ করুন।
৩. আপনি চাইলে মূল ডোমেইনে বা একটি সাব-ফোল্ডারে (যেমন: `public_html/admin/` অথবা `public_html/api/`) এই স্ক্রিপ্টগুলো রাখতে পারেন। 
৪. এই `/server-script/` ফোল্ডার থেকে নিচের ২টি ফাইল আপনার হোস্টিং-এ আপলোড করুন:
   - `api.php`
   - `admin.php`

### ধাপ ২: ডাটা ফাইল ও পারমিশন নিশ্চিত করা
* প্রথমবার ব্রাউজারে `api.php` বা `admin.php` ফাইলটি রান করালেই অটোমেটিক একটি `data.json` ফাইল তৈরি হয়ে যাবে।
* ফাইল ম্যানেজারে নিশ্চিত করুন যেন আপনার ডিরেক্টরিতে রাইট পারমিশন (Write Permission) থাকে, যাতে এডমিন প্যানেল থেকে ডাটা সেভ করলে তা `data.json` ফাইলে আপডেট হতে পারে (সাধারনত ডিফল্টভাবেই পারমিশন দেওয়া থাকে)।

### ধাপ ৩: পাসওয়ার্ড পরিবর্তন করা (নিরাপত্তার জন্য)
১. আপনার হোস্টিং-এর ফাইল ম্যানেজার থেকে `admin.php` ফাইলটি এডিট করুন।
২. ফাইলের একদম শুরুর দিকে নিচের লাইনগুলো দেখতে পাবেন:
   ```php
   // Admin Authentication Configuration
   define('ADMIN_USER', 'admin');
   define('ADMIN_PASS', '123456'); // <--- এখানে আপনার নিজস্ব কঠিন পাসওয়ার্ড বসিয়ে দিন!
   ```
৩. `123456` পরিবর্তন করে আপনার গোপন মনমতো পাসওয়ার্ড বসিয়ে ফাইলটি সেভ করে দিন।

### ধাপ ৪: ব্রাউজারে এডমিন প্যানেল চেক করা
* এখন ব্রাউজারে আপনার ডোমেন ভিজিট করুন:
  - যদি মূল ফোল্ডারে রাখেন: `https://yourdomain.com/admin.php`
  - যদি আলাদা `admin` ফোল্ডার করে তার ভেতর রাখেন: `https://yourdomain.com/admin/admin.php`
* এডমিন প্যানেলে ইউজারনেম `admin` এবং আপনার সেট করা পাসওয়ার্ড দিয়ে লগইন করুন।
* এখান থেকে আপনি খুব সহজেই **লাইভ ম্যাচ**, **টিভি চ্যানেল** এবং **নোটিশ বা বিজ্ঞাপন** যুক্ত বা পরিবর্তন করতে পারবেন!

## 📱 অ্যান্ড্রোয়েড অ্যাপের সাথে যুক্ত করার নিয়ম (Android Sync Setup)

১. আপনার অ্যান্ড্রোয়েড অ্যাপে প্রবেশ করুন।
২. অ্যাপের ওপরের ডান কর্নারে থাকা **Admin Panel** বাটনে ক্লিক করে পাসওয়ার্ড দিয়ে প্রবেশ করুন।
৩. স্ক্রল করে নিচের দিকে **অনলাইন ডাটাবেস সিঙ্ক (Server Sync)** কার্ডে যান।
৪. সিঙ্ক বাটনটি **অন (Enable)** করুন।
৫. আপনার হোস্টিং-এর API URL টি প্রবেশ করান। যেমন:
   `https://yourdomain.com/api.php` (অথবা আপনার কাস্টম পাথ)
৬. **সেটিংস সেভ** বাটনে ক্লিক করুন।
৭. এরপর **সিঙ্ক করুন** বাটনে চাপ দিন। সাথে সাথে আপনার অনলাইন সার্ভার থেকে সকল ম্যাচ এবং চ্যানেলের ডাটা সেকেন্ডের মধ্যে খেলাঘর অ্যান্ড্রোয়েড অ্যাপে সিঙ্ক হয়ে চলে আসবে!

### 💡 সুবিধা সমূহ:
* **কোনো ডাটাবেস লাগবে না**: এটি MySQL ডাটাবেস ছাড়াই সম্পূর্ণ লাইটওয়েট ডিক্লেয়ারেশনে রান করে। ফলে খুব সস্তার হোস্টিং-এও এটি অবিশ্বাস্য স্পিডে কাজ করবে।
* **রুম অফলাইন ক্যাশ সাপোর্ট**: অ্যাপ একবার আপনার সার্ভার থেকে সিঙ্ক করে ডাটা নিয়ে নিলে পরবর্তীতে ইউজারদের নেটওয়ার্ক না থাকলেও বা সার্ভার সাময়িক ডাউন থাকলেও আগের লোড হওয়া খেলাগুলো সুন্দরভাবে অফলাইনেও দেখতে পাবে (Room database implementation-এর মাধ্যমে এটি সম্পন্ন হয়)।
