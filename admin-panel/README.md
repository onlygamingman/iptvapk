# KhelaGhor - Web Admin Panel & API Guide
(Self-Hosted MySQL, Secured prepared statement APIs & Responsive Dashboards)

এই ডিরেক্টরিতে আপনার কেনা নিজস্ব ডোমেইন এবং হোস্টিং সাইটে KhelaGhor অ্যাপের সেন্ট্রাল কন্ট্রোল ডেক্স সেটআপ দেওয়ার সম্পূর্ণ সিকিউরড পিএইচপি ফাইলসমূহ (Web Admin Panel) এবং ডেটাবেজ স্কিমেটিক্স সংকলিত রয়েছে। 

---

## ১. মূল ফাইলসমূহ এবং কার্যাবলী (File Directory Summary)

1. **`db.sql`**: আপনার হোস্টিং MySQL ডেটাবেজে রান করানোর মূল স্ট্রাকচার ফাইল। এটিতে প্রয়োজনীয় ইনডেক্সিং এবং ডিফল্ট অ্যাডমিন কুয়েরি ডেটা হাসিং সেট করা আছে।
2. **`config.php`**: পিএইচপি ডাটাবেজ কানেকশন কনফিগারেশন ফাইল। এতে সম্পূর্ণ হ্যাকিং-প্রুফ **PDO (PHP Data Objects)** সংযোগ ব্যবস্থা করা হয়েছে, যা SQL Injection (SQLi) সম্পূর্ণরূপে প্রতিরোধ করে।
3. **`index.php`**: একটি সম্পূর্ণ রেসপনসিভ ডার্ক ফরেস্ট গ্রিন থিমযুক্ত অ্যাডমিন ড্যাশবোর্ড। এই সিঙ্গেল-প্যানেল ড্যাশবোর্ডে আপনি পিসি, ল্যাপটপ কিংবা যেকোনো মোবাইল ব্রাউজার (যেমন Chrome) থেকে ম্যাচ কার্ড কন্ট্রোল করতে পারবেন।
4. **`api.php`**: অ্যান্ড্রয়েড অ্যাপের সাথে ডেটা সিকিউর আদানপ্রদান করার REST API যা স্বয়ংক্রিয়ভাবে JSON ফরম্যাটে লাইভ ফিডস, চ্যানেল ফোল্ডার এবং বিজ্ঞাপন সেটিংস রিটার্ন করবে।

---

## ২. ডোমেইন ও হোস্টিং সেটআপ গাইডলাইন (Local/cPanel Web Server Setup)

### ধাপ ক: ডাটাবেজ তৈরি এবং ইম্পোর্ট
1. আপনার হোস্টিংয়ের **cPanel**-এ লগইন করুন এবং **MySQL Database Wizard**-এ ক্লিক করে নতুন ডাটাবেজ খুলুন (যেমন: `khelaghor_db`)।
2. নতুন একটি ডাটাবেজ ইউজার তৈরি করুন এবং উক্ত ইউজারের সাথে ডাটাবেজটি কানেক্ট করে সমস্ত প্রিভিলেজ (All Privileges) অন করে দিন।
3. এবার cPanel-এর হোমপেজ থেকে **phpMyAdmin** ওপেন করুন।
4. বাম পাশে আপনার তৈরি করা নতুন ডাটাবেজে ক্লিক করে ওপরের **Import** ট্যাবে যান।
5. আপনার এই প্রোজেক্ট থেকে ডাউনলোডকৃত **`db.sql`** ফাইলটি সিলেক্ট করে নিচে স্ক্রোল করে **Go** বাটনে ক্লিক করুন। মুহূর্তের মধ্যে সমস্ত টেবিল তৈরি হয়ে যাবে।

### ধাপ খ: পিএইচপি ফাইল আপলোড ও ডাটাবেজ সংযোগ
1. cPanel থেকে **File Manager** ওপেন করে আপনার ডোমেইনের ডিরেক্টরি ফোল্ডারে যান (যেমন: `public_html/admin/`)।
2. সেখানে আমাদের এই অ্যাডমিন ডিরেক্টরির ৪টি মূল ফাইল (`config.php`, `index.php`, `api.php`, `db.sql`) আপলোড করুন।
3. ফাইল আপলোড হয়ে গেলে, **`config.php`** ফাইলটি এডিট মোডে ওপেন করুন এবং নিচের লাইনগুলো আপনার পূর্বে তৈরি ডাটাবেজ ক্রেডেনশিয়াল দিয়ে পরিবর্তন করে দিন:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_USER', 'আপনার_ডাটাবেজ_ইউজার');
   define('DB_PASS', 'আপনার_ডাটাবেজ_পাসওয়ার্ড');
   define('DB_NAME', 'আপনার_ডাটাবেজ_নাম');
   ```
4. ফাইলটি সেভ করে বের হয়ে যান।

## ৩. ড্যাশবোর্ড এক্সেস এবং সিকিউরিটি সেটিংস (Usage & Security Protocols)

- কাস্টম ব্রাউজারে `yourdomain.com/admin` হিট করুন।
- এখানে একটি ৩-পর্যায়ের ব্রুট-ফোর্স প্রটেকশন সংবলিত ভেরিফিকেশন লগইন পেজ দেখতে পাবেন।
- **ডিফল্ট পাসওয়ার্ড:** `admin123`
- আপনি লগইন করার পর ড্যাশবোর্ডের ৩য় ট্যাব **"বিজ্ঞাপন ও পাসওয়ার্ড"**-এ গিয়ে আপনার মালিকানাধীন কাস্টম নিউ সিকিউরড পাসওয়ার্ডটি সেভ করে ফেলুন। এটি ডেটাবেজে সরাসরি ওয়ান-ওয়ে **Bcrypt (Cryptographic Hash Algorithm)** উপায়ে এনক্রিপ্ট হয়ে সেভ হবে, যা হ্যাকিং শতভাগ প্রতিরোধ করবে।

## ৪. অ্যান্ড্রয়েড অ্যাপের সাথে এপিআই কানেক্ট করার নিয়ম (API Sync Setup)

অ্যান্ড্রয়েড অ্যাপে লোকাল রুম ডাটাবেজ ব্যবহারের সাথে সাথে আপনার দূরবর্তী হোস্টেড সাইটের ডেটা লোড করাতে চাইলে আপনি অ্যাপের `SportsRepository.kt` অথবা এপিআই কল করার মেথডে আপনার Hosted ডোমেইন এপিআই লিংকের সাথে সিঙ্ক করতে পারবেন।

- **লাইভ ম্যাচ কুয়েরি লিংক:** `https://yourdomain.com/admin/api.php?action=matches`
- **লাইভ টিভি চ্যানেল কুয়েরি লিংক:** `https://yourdomain.com/admin/api.php?action=channels`
- **গ্লোবাল বিজ্ঞাপন ট্রিগার কুয়েরি লিংক:** `https://yourdomain.com/admin/api.php?action=config`

## ৫. হ্যাকিং প্রতিরোধে মূল সাইবার নিরাপত্তা সম্বলিত গাইডলাইন (Advanced Cyber Security)

1. **HTTPS (SSL) বাধ্যতামূলক ব্যবহার:** আপনার সার্ভারে সবসময় SSL নিশ্চিত করুন যেন RESTful API এবং অ্যাডমিন লগইন ক্রেডেনশিয়াল ট্রাফিকের মধ্যে এনক্রিপ্টেড উপায়ে যাতায়াত করে।
2. **SQL Injection (SQLi) থেকে সুরক্ষা:** আমরা কোডে সরাসরি $pdo->prepare() এবং কোয়েশ্চেন মার্ক (?) প্যারামিটারাইজিং ব্যবহার করেছি। কখনোই পিএইচপি-তে কোনো Raw Query বা ডেটা স্যানিটাইজিং ছাড়া ইনপুট ফিল্ড ডাইরেক্ট কুয়েরিতে পাঠাবেন না।
3. **লগইন রেট লিমিটিং:** পরপর ৩ বার ভুল পাসওয়ার্ড ইনপুট দিলে সেশন রেট লিমিটিং ট্রিগার হবে যা সাময়িক সময়ের জন্য ইউজার আইপি রিডাইরেক্টিং ডেনাইড করে দেবে।
